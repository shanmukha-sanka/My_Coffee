# Coffee Bloom ☕

A single-origin coffee shop website, founded by Shanmukha Sanka in Vijayawada, India.

🔗 **Live site:** https://shanmukha-sanka.github.io/My_Coffee/

## About
Coffee Bloom sources coffee from volcanic-soil growing regions — Guatemala, Ethiopia, and Java — 
roasting every batch in small drums and tracking the roast curve by hand.
